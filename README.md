\# File Organizer Python Script

\## كيفية الاستخدام

1\. افتح `file\_organizer.py` وعدل المسار `source\_folder` ليكون المجلد الذي تريد ترتيب الملفات فيه.

2\. شغل السكريبت:

```bash

python file\_organizer.py
